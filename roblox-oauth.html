Roblox profile connection for Lockou Admin

This page explains the Roblox OAuth connection used by Lockou Admin. The connection is used only to link an approved admin account with the Roblox profile that owns it and to load the Roblox avatar image inside the admin sidebar

Scopes: openid profile Purpose: account linking Data shown: avatar and basic profile

What this app does

Links a Roblox profile to an approved admin account Used to identify the Roblox profile tied to that admin account
Loads the Roblox avatar icon The avatar image is shown in the private admin sidebar after authorization
Stores only basic profile fields Roblox user ID, username, display name, profile link and avatar image URL

What this app does not do

It does not post on behalf of the user No social actions or game actions are performed with the connected account
It does not request more than profile access The app requests only the profile data needed for the visual identity of the admin
It does not open public access to the admin panel The admin workspace remains private and protected by internal access approval
