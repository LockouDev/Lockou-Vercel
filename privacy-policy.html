Privacy Policy

How Lockou Admin handles data

This Privacy Policy explains how Lockou Admin collects, stores and uses data related to private admin access and the optional Roblox OAuth profile connection

Last updated: March 11, 2026 Owner: Lockou Contact: kaiojeffoficial@gmail.com

Data we collect

  • Admin username and encrypted password data for private access control
  • Admin role, permissions, approval status and security session records
  • Roblox profile data only when the admin authorizes the Roblox OAuth request
  • When authorized, the stored Roblox profile data may include Roblox user ID, username, display name, avatar image URL and profile URL

How we use data

  • To protect access to the private Lockou Admin workspace
  • To approve or reject admin requests and assign internal permissions
  • To show the connected Roblox avatar image and basic profile identity inside the private admin sidebar
  • To operate Roblox DataStore lookup and internal admin tools

How data is stored

  • Admin account data and session data are stored in infrastructure configured for the Lockou Admin project
  • Roblox OAuth profile data is stored only after the user grants permission
  • Data is retained for as long as it is needed to operate the private admin workspace or until it is manually removed

How data is shared

  • Data is not sold
  • Data is not shared publicly
  • Data may be processed through service providers required to run the website and secure the admin system
  • Roblox profile data is used only for the private admin account that authorized the connection

Your choices

  • Roblox OAuth profile connection is optional for the admin account
  • If the user denies the Roblox OAuth request, the admin account continues without the avatar image
  • Questions or requests about stored data can be sent to kaiojeffoficial@gmail.com
This policy applies to the Lockou Admin website and the Roblox OAuth profile connection used by approved admin accounts
